# Credit Risk Assessment System

**Developer:** Nasisipho  
**Institution:** Belgium Campus ITversity

## Project Overview

An AI-powered dashboard that predicts the probability of loan default using XGBoost and K-Means clustering.

## Deployment Instructions

1. **Environment Setup:** - Ensure Python 3.x is installed.
   - Install dependencies: `pip install -r requirements.txt`
2. **Running Locally:**
   - Run `python app.py`
3. **Artifacts:**
   - Ensure `.pkl` models are located in the `/artifacts` folder before deploying.

## Tech Stack

- **Frontend:** Dash (Plotly), Dash Bootstrap Components
- **Styling:** Custom CSS (Cyber Grape Theme)
- **Deployment:** Gunicorn / Heroku
